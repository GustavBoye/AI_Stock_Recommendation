Multi-Horizon Stock Prediction Engine

A deep learning–based multi-horizon stock forecasting system built with TensorFlow, NumPy, and Yahoo Finance (yfinance).

This project downloads historical market data for a large portfolio of stocks, trains convolutional neural network ensembles across multiple forecast horizons, and generates visual prediction dashboards for each ticker.

Developed by Gustav Morving
www.gustav-morving.dk

Features
Multi-horizon forecasting (32d, 64d, 76d, 90d, 120d)
Ensemble deep learning models
Uses:
Close prices
Volume
High prices
Low prices
Automatic data normalization
CNN-based forecasting architecture
Inverted-data augmentation
Prediction export to CSV
Dark-mode prediction visualization dashboards
Batch Yahoo Finance downloader
Adaptive learning rate scheduling
Early stopping support
